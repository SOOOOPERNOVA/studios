리액티브 바이럴 콘텐츠 기획·제작

역할
기획 · 편집
유형
극장 애니메이션 개봉 프로모션 (바이럴 · 홍보 영상)
비고
영상 원본은 권리 관계상 비공개. 요청시 개별 열람 가능.

Problem

짧은 개봉 기간 안에 광고 노출이 아닌 자발적 확산이 필요했던 상황

Solution

타이밍과 화제성으로 접근

  • 실시간 이슈 캐치 — 개봉 시기에 맞춰 발생한 도심 담비 출몰 뉴스를 즉시 포착, 작품의 주인공 캐릭터와 연결해 콘텐츠화
  • 포맷 차용 디자인 — 뉴스 리포트 형식의 그래픽·자막 시스템을 직접 설계해 캐릭터가 실제 뉴스에 등장한 듯한 연출 구현
  • 동시기 화제작 활용 — 같은 시기 극장가를 지배하던 흥행작의 화법과 톤을 차용, 관객이 이미 익숙한 문법 위에 작품을 얹어 진입 장벽을 낮춤
  • 밈 기반 바이럴 — 온라인에서 이미 소비 문법이 형성된 밈 포맷을 차용, 짧은 러닝타임의 확산형 콘텐츠 다수 제작
